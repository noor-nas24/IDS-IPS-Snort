## **Snort Operation Modes**
1. Sniffer Mode --> which simply reads the packets off of the network and displays them for you in a continuous stream on the console.
   
2. Packet Logger Mode--> collect and logs network paket in file.
   
3. Network Intrusion Detection System (NIDS) mode--> which performs detection and analysis on network traffic. This is the most complex and configurable mode.
